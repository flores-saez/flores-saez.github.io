# alonso-flores-saez.github.io

Personal portfolio and research site for Alonso Flores-Sáez — social science researcher, data analyst, Philadelphia PA.

## Pages

| File | Description |
|---|---|
| `index.html` | Home — hero, about, research, experience, contact |
| `work.html` | Full experience page — professional, research, advocacy, education, skills |
| `research-disability.html` | Research: Disability and Care in Underrepresented Communities (Asdown Colombia) |
| `research-alce.html` | Research: Involuntary Institutionalization in Colombia (A.L.C.E.) |
| `research-thesis.html` | Research: Digital Technology Use and Community Identification (UC Irvine M.A. Thesis) |

## Stack

Plain HTML, CSS, and vanilla JavaScript — no frameworks, no dependencies, no build step.

## Deployment

Hosted via GitHub Pages. To deploy, push to the `main` branch and ensure GitHub Pages is set to serve from the root of `main` in repository Settings → Pages.
